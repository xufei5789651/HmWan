## 玩Android HarmonyOS 版
### 简介
玩Android 采用ArkTS语言编写，采用分层架构+模块化等架构，进行一多适配

### 功能
- 主页
- 项目
- 登录/注册
- 我的分享
- 我的收藏
- 深色模式
- 浏览文章、分享文章
- 我的积分、积分排行榜
- 我的消息
